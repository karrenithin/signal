# 📱 Signal

Signal is a modern mobile buying application that helps users discover, compare, and purchase smartphones with ease. Built with a clean and responsive design, Signal provides a seamless shopping experience for users looking for the latest mobile devices.

## 🚀 Features

- Browse smartphones from top brands
- Advanced search and filtering
- Product comparison
- Wishlist functionality
- User ratings and reviews
- Secure checkout process
- Order tracking
- Responsive UI for all devices

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
